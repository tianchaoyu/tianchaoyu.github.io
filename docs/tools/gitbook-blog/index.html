    如何更新github博客

    目录

    • jekyll本地文件编辑
    • github本地文件备份与上传

    jekyll本地文件编辑

    • 参考jekyll格式，进行本地文件编辑
    • 配置好相关文件
      • 配置_config.yml文件下scope目录，collections；
      • 配置_data目录，_data/navigation.yml文件,每个知识库的链接
      • 指定本地化文件输出路径：_config.yml->destination: docs
    • 用jekyll进行本地化编译
      • bundle exec jekyll build
    • 启动本地服务进行查看
      • bundle exec jekyll serve

    github本地文件备份与上传

    • 在待上传的目录外面，git init初始化git仓库
    • 添加更改到缓行区，git add docs
    • 提交更改到本地仓库：git commit -m “update”
    • 推送更改到远程仓库：git push origin master
    • 强制推送：git push origin main:master –force
