    • 总结

    出行链生成模型是研究人员出行链产生的总量模型，是交通需求分析工作中最基本的部分。 出行链的发生受社会经济、人口岗位、土地利用、机动车发展水平等多种因素的影响。

    0、基本定义

    • 活动（activity）：个体为了满足自身或家庭在经济方面的、生理方面的和社会等方面的需要，而在某件事情中的实体参与；
    • 出行（trip)：基于活动目的，采用一种或多种交通方式从一个地点到另一个地点的过程，例如：家-单位的上班出行（HB Work）、购物-家的购物回家（HB Shop）、单位-客户（FB Business）等；
    • 活动链（activity chain）:以24小时为单位，将出行者一系列活动按照先后顺序依次连接起来所形成的链条。
    • 出行链（tour)：从起点出发, 经过若干中途停留点后, 最终返回起点的一连串出行组合，如家-单位-家（HWH）、单位-餐厅-单位（FEF）等；
    • 基家（HB tour）& 非基家（NHB tour）出行链：起终点都是家的出行链；起终点不是家的出行链为非基家出行链。
    • 主要停留点（PD，Primary Destination）：指优先级最高（如通勤）或耗时最长或距离起点最远的活动的中间停留点。
    • 次要停留点（SD，Secondary Destination ）：指PD以外的中间停留点，如Home-Escort-Work-Home中的Escort。

    • 简单链：仅有一个中间停留点的出行链，如HWH；
    • 复杂链：中间含有两个及以上的中间停留点的出行链，例如：HWSH(家-工作-购物-家),HCWH(家-送孩子上学-工作-家）；
    • 去程（前半程）：出行链起点到主要停留点的半程，例如：HWH的HW；HCWH的主要停留点为work,前半程链为HCW。
    • 回程（后半程）：出行链主要停留点回到起点的半程，例如： HWH的WH；HWSH的主要停留点为work,后半程链为WSH。
    • 半程（half tour)：24小时内无法回到起点的tour，不含起点，仅1个停留点;
    • 基本出行链：从调查的出行链中归并、筛选后，用于分析与出行链建模的出行链。

    2、出行链理论

    出行链目的类型参考 基于家的出行

    • 基于家的工作再回家（HWH，Home-Work-Home） 以家为起点出发去工作，在当天工作结束后从工作地点返回家的出行链。
    • 基于家的上小学再回家（HEH，Home-Education-Home） 以家为起点出发去上小学，在当天学习结束后从学校返回家的出行链。
    • 基于家的上中学再回家（HMH，Home-MiddleSchool-Home） 以家为起点出发去上中学，在当天学习结束后从学校返回家的出行链。
    • 基于家的上大学再回家（HUH，Home-University-Home） 以家为起点出发去高校学，在当天学习结束后从学校返回家的出行链。
    • 基于家的商务出行再回家（HBH，Home-Bussiness-Home） 以家为起点的商务出行，在当天办公结束后返回家的出行链。
    • 基于家的购物再回家（HSH，Home-Shopping-Home） 以家为起点出发去购物，在购物结束后从购物地点返回家的出行链。
    • 基于家的其它出行再回家（HOH，Home-Other-Home） 以家为起点出发进行其它类型的出行，在当天的出行结束后，从出行的目的地返回家的出行链。

    • 基于家的送小孩再回家（HCH，Home- Escort -Home） 以家为起点出发去送小孩，在送孩子上学后返回家的出行链。以非家的地点作为起点出行，在当天的出行结束后，从出行目的地返回非家的出行起点，目前武汉宏观模型主要考虑的是基于工作地的出行。

    • 基于工作地的外出吃饭再返回工作地（FAF，Office-Eatingout -Office） 以工作地为起点，外出吃饭，在吃完饭后，返回工作地点的出行链。
    • 基于工作地的商务出行，再返回工作地（FBF，Office-Business -Office） 以工作地为起点，外出商务办公，在办公结束后，返回工作地点的出行链。

    • 基于工作地的其他出行，再返回工作地（FOF，Office- Other -Office） 以工作地为起点的其他类型的出行，当天结束后，返回工作地点的出行链。除了以上基于一个目的地的出行链，武汉tour base model还考虑了多个目的地的出行链，主要有：

    • 基于家的送孩子上学、去工作再回家（HCWH，Home-Escort-Work-Home） 以家为起点先送孩子上学，将孩子送到学校后，再去工作地点工作，在当天工作结束后从工作地点返回家的出行链，其中工作地作为主要目的地，学校作为家端到工作地端中间的短暂停留点。
    • 基于家的去工作、外出吃饭再回家（HWAH，Home-Work-Eat-Home） 以家为起点先去工作地点工作，在当天工作结束后，从工作地点出发，去饭店等餐饮场所吃饭，就餐结束后返回家的出行链，工作地作为主目的地，外出吃饭点作为工作端到家端的中间停留点。## 3、交叉类别 人群和出行目的交叉分类中，实际上有很多人群不具备某种出行，可手动调整。
