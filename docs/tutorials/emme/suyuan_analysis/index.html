    1、功能

    分析关键通道，出行需求来源。

    2、实现方法

    1）创建额外标记属性（默认参数为0），选择分析对象，将额外参数赋值为1； 2）使用道路交通分配方法，默认使用sola traffic assignent; 3）根据分析类型（车辆类型），选择需求分析矩阵； 4）路径分析：统计path上额外属性值的总和，根据阈值统计该路径上的出行需求矩阵； 5）结果输出与分析；
